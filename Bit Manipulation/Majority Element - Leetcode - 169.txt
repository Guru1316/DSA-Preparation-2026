169 - Majority Element

Given an array nums of size n, return the majority element.

The majority element is the element that appears more than ⌊n / 2⌋ times. You may assume that the majority element always exists in the array.

 

Example 1:

Input: nums = [3,2,3]
Output: 3
Example 2:

Input: nums = [2,2,1,1,1,2,2]
Output: 2
 

Constraints:

n == nums.length
1 <= n <= 5 * 104
-109 <= nums[i] <= 109
The input is generated such that a majority element will exist in the array.


Solution in CPP:

class Solution {
public:
    int majorityElement(vector<int>& nums) {
        int c = 0;
        int el;
        for(int n : nums)
        {
            if(c==0)
            {
                c = 1;
                el = n;
            }
            else if(el == n)
            {
                c++;
            }
            else
            {
                c--;
            }
        }
        c = 0;
        for(int n : nums)
        {
            if(n == el)
            {
                c++;
            }
        }
        if(c > nums.size()/2)
        {
            return el;
        }
        return -1;
    }
};

Explanation:

first set c(count) = 0
then declare el variable

then run loop for traversing over the array

write 3 conditions
i) if c == 0 then set c = 1 and set current value of array to el
ii) if el == current value means then increment c by 1
iii) else decrement c by 1

after coming out of the loop make c = 0

then again run the loop for traversing over the array
here check one condition
if current value == el means increment c

after coming out of loop check whether c > nums.size / 2

if yes return el
else return -1

Example 1:

Input: nums = [3,2,3]

1) current value - 3 
   c = 0 so now c = 1 and el = 3

2) current value - 2
   el != current value so c--, so  c = 0

3) current value - 3  
   c = 0 so now c = 1 and el = 3

comes out of loop

next loop

now c = 0

1) cur val = 3 == el -> c = 1
2) cur val = 2 != el -> c = 1
3) cur val = 3 == el -> c = 2

c > nums.size()/2 => 2>3/2 => 2>1

so el is the answer (i.e) 3 is the answer

Here only majority element survives till last no other element can survive because majority element is greater than half of array size
